# React Authentication UI

A modern authentication UI built with React. This project includes Login and Register pages with a clean glassmorphism design, form validation, and reusable form components.

## Features

- Login Page
- Register Page
- Responsive Design
- Glassmorphism UI
- Form Validation with Formik & Yup
- Reusable Form Components
- React Router Navigation
- Bootstrap 5 Styling
- RTL (Persian) Support

---

## Technologies

- React
- React Router DOM
- Bootstrap 5
- React Bootstrap
- Formik
- Yup

---

## Packages Used

| Package | Purpose |
|---------|---------|
| react | UI Library |
| react-dom | React DOM Rendering |
| react-router-dom | Client-side Routing |
| bootstrap | CSS Framework |
| react-bootstrap | Bootstrap Components for React |
| formik | Form State Management |
| yup | Form Validation |
| react-icons | Icons |

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

Go to the project folder

```bash
cd your-repository
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```
